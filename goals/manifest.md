# Goals Manifest

Index of available goal workflows in the OmniChat system.

| Goal File                | Trigger                                      | Description                                                   |
|--------------------------|----------------------------------------------|---------------------------------------------------------------|
| voice_conversation.md    | User speaks via microphone or types a message | Audio/text in → model inference → audio/text response out     |
| session_playback.md      | User selects a recorded session for review    | Session manifest/video in → transcript replay, overlay, export |
